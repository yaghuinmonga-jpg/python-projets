# Projet 1 – Simulation de jeux de hasard en Python

## Objectif
Ce projet vise à simuler un jeu de hasard à l’aide du langage Python afin d’analyser :
- l’évolution du capital d’un joueur
- la notion d’espérance mathématique
- la variance et le risque de ruine

## Description
Un joueur démarre avec un capital initial et participe à un jeu où il peut gagner ou perdre une mise fixe à chaque partie.  
Le jeu est répété un grand nombre de fois afin d’observer le comportement du capital sur le long terme.

## Outils utilisés
- Python
- random
- numpy
- matplotlib

## Résultats principaux
- L’espérance de gain par partie est proche de zéro (jeu théoriquement équitable)
- La variance est élevée, ce qui entraîne une forte volatilité du capital
- Même avec une espérance nulle, le risque de pertes importantes existe

## Conclusion
Ce projet montre que l’espérance seule ne suffit pas à évaluer un jeu de hasard.  
La variance et le risque de ruine jouent un rôle central dans l’analyse des jeux aléatoires.
