# Projet 2 — Analyse de l’inflation en France (INSEE)

## Objectif
Analyser l’évolution du taux d’inflation en France à partir de données mensuelles (source INSEE) et réaliser :
- un nettoyage simple des données
- des statistiques descriptives
- une visualisation de l’évolution sur la période 1991–2025

## Données
- Source : INSEE (indice des prix à la consommation, glissement annuel)
- Fréquence : mensuelle
- Période étudiée : 1991 à 2025

## Étapes principales
1. Import du fichier CSV
2. Nettoyage : sélection des lignes utiles, conversion des types (date + valeurs numériques)
3. Statistiques descriptives (describe)
4. Graphique de l’évolution du taux d’inflation
5. Interprétation simple : périodes marquantes (2008, 2020–2022)

## Outils utilisés
- Python
- pandas
- matplotlib

## Résultat
Le graphique met en évidence une inflation globalement modérée sur le long terme, avec des épisodes de variations plus fortes, notamment autour de la crise financière de 2008 et de la période Covid / post-Covid (2020–2022).
